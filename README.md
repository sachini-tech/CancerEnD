# CancerEnD: Cancer Associated Enhancer Database

CancerEnD is a comprehensive and user-friendly database developed for the analysis of cancer-associated enhancer elements and their role in tumorigenesis. The database integrates enhancer expression, associated genes, somatic mutations, copy number variations (CNVs), and survival information across multiple cancer types, providing an important resource for non-coding cancer genomics research.

## About the Database

CancerEnD was developed to systematically annotate enhancer elements involved in different cancer types. Enhancers are important non-coding regulatory elements that control tissue-specific gene expression, and their dysregulation can contribute to cancer initiation and progression.

The database integrates large-scale genomic and epigenomic datasets from TCGA, COSMIC, and FANTOM to provide detailed information about enhancer-associated alterations in cancer.

CancerEnD contains:

- **8524 unique expressed enhancers**
- **8063 cancer samples**
- **18 cancer types**
- **1762 survival-associated enhancers**
- Somatic mutation and CNV information for enhancer regions
- Enhancer-gene interaction data

---

## Key Features

- Cancer-specific enhancer annotations
- Somatic mutation mapping within enhancer regions
- Copy number variation analysis of enhancers
- Survival-associated enhancer identification
- Enhancer-gene association analysis
- Interactive search and browse modules
- Downloadable datasets for further analysis

---

## Applications

CancerEnD can be used for:

- Cancer genomics and epigenomics research
- Identification of prognostic biomarkers
- Analysis of non-coding mutations in cancer
- Enhancer-based therapeutic target discovery
- Survival analysis studies
- Understanding enhancer deregulation in tumorigenesis
